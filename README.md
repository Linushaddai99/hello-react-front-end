### Setup

1. Clone the repo by running the command
   ```sh
    git clone https://github.com/linushaddai99/hello-react-front-end.git
   ```
2. Open the directory of the project
   ```sh
   cd hello-react-front-end
   ```
3. Install project dependencies
   ```sh
   run npm install
   ```
